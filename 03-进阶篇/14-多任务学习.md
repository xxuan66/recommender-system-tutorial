# 第十四章：多任务学习

> 当业务目标不止一个时，模型为什么也不能只学一个任务

---

## 14.1 为什么需要多任务学习

### 14.1.1 推荐系统的多目标性

**在真实业务里，往往不只关心一个指标：**

| 场景 | 目标 1 | 目标 2 | 目标 3 |
|------|-------|-------|-------|
| **电商** | 点击率 | 转化率 | GMV |
| **视频** | 点击率 | 观看时长 | 完播率 |
| **新闻** | 点击率 | 阅读时长 | 分享率 |
| **广告** | 点击率 | 转化成本 | ROI |

### 14.1.2 单任务 vs 多任务

**单任务学习容易出现的问题：**

```
单独优化 CTR → 标题党内容泛滥
单独优化时长 → 低质长内容增多
单独优化转化 → 只推荐低价商品
```

**多任务学习的优势：**

```
同时优化 CTR + 时长 + 转化 → 平衡内容质量和商业价值
```

### 14.1.3 多任务的挑战

| 挑战 | 说明 | 解决方案 |
|------|------|---------|
| **任务冲突** | 不同目标可能矛盾 | 梯度协调 |
| **数据不平衡** | 正样本比例不同 | 损失加权 |
| **模型复杂** | 参数量大 | 参数共享 |

## 14.2 多任务学习方法

### 14.2.1 发展脉络

```
2018: MMOE (Multi-gate Mixture-of-Experts)
       ↓
2019: PLE (Progressive Layered Extraction)
       ↓
2020: AITM (Adaptive Information Transfer Multi-task)
       ↓
2021: D-MoE (Distributed MoE)
```

### 14.2.2 方法对比

| 方法 | 核心思想 | 优点 | 缺点 |
|------|---------|------|------|
| **Shared-Bottom** | 共享底层参数 | 简单 | 任务冲突 |
| **MMoE** | 多专家混合 | 缓解冲突 | 专家可能退化 |
| **PLE** | 分离专家 | 更好解耦 | 参数更多 |
| **AITM** | 自适应迁移 | 动态调整 | 复杂度高 |

## 14.3 Shared-Bottom

### 14.3.1 架构

```
┌─────────────────────────────────────────────┐
│   任务 1 塔    │   任务 2 塔    │   任务 3 塔    │
│   (CTR)       │   (CVR)       │   (时长)      │
└───────┬───────┘   └───────┬───────┘   └───────┬───────┘
        │                   │                   │
        └───────────────────┼───────────────────┘
                            │
                    ┌───────▼───────┐
                    │   共享底层     │
                    │  (Shared)     │
                    └───────┬───────┘
                            │
                    ┌───────▼───────┐
                    │    嵌入层      │
                    └───────────────┘
```

### 14.3.2 代码实现

```python
import torch
import torch.nn as nn

class SharedBottomMTL(nn.Module):
    """Shared-Bottom 多任务模型"""
    
    def __init__(
        self,
        n_users,
        n_items,
        embedding_dim=16,
        shared_hidden_dims=[128, 64],
        task_hidden_dims=[32],
        n_tasks=3,
    ):
        super(SharedBottomMTL, self).__init__()
        
        self.n_tasks = n_tasks
        
        # 嵌入层
        self.user_embedding = nn.Embedding(n_users, embedding_dim)
        self.item_embedding = nn.Embedding(n_items, embedding_dim)
        
        # 共享底层
        shared_layers = []
        input_dim = embedding_dim * 2
        
        for hidden_dim in shared_hidden_dims:
            shared_layers.append(nn.Linear(input_dim, hidden_dim))
            shared_layers.append(nn.ReLU())
            shared_layers.append(nn.BatchNorm1d(hidden_dim))
            input_dim = hidden_dim
        
        self.shared_network = nn.Sequential(*shared_layers)
        
        # 任务塔（每个任务一个）
        self.task_towers = nn.ModuleList()
        
        for _ in range(n_tasks):
            tower_layers = []
            tower_input = shared_hidden_dims[-1] if shared_hidden_dims else embedding_dim * 2
            
            for hidden_dim in task_hidden_dims:
                tower_layers.append(nn.Linear(tower_input, hidden_dim))
                tower_layers.append(nn.ReLU())
                tower_input = hidden_dim
            
            # 输出层
            tower_layers.append(nn.Linear(tower_input, 1))
            
            self.task_towers.append(nn.Sequential(*tower_layers))
    
    def forward(self, user_ids, item_ids):
        # 嵌入
        user_emb = self.user_embedding(user_ids)
        item_emb = self.item_embedding(item_ids)
        concat = torch.cat([user_emb, item_emb], dim=1)
        
        # 共享底层
        shared_repr = self.shared_network(concat)
        
        # 多个任务输出
        task_outputs = []
        for tower in self.task_towers:
            output = tower(shared_repr)
            task_outputs.append(torch.sigmoid(output).squeeze())
        
        return task_outputs  # [task1_pred, task2_pred, ...]


# 使用示例
model = SharedBottomMTL(
    n_users=1000,
    n_items=500,
    embedding_dim=16,
    shared_hidden_dims=[128, 64],
    task_hidden_dims=[32],
    n_tasks=3  # CTR, CVR, 时长
)

user_ids = torch.LongTensor([0, 1, 2])
item_ids = torch.LongTensor([10, 20, 30])

outputs = model(user_ids, item_ids)
print(f"任务 1 输出：{outputs[0]}")
print(f"任务 2 输出：{outputs[1]}")
print(f"任务 3 输出：{outputs[2]}")
```

### 14.3.3 损失函数

```python
class MultiTaskLoss(nn.Module):
    """多任务损失"""
    
    def __init__(self, task_weights=None):
        super(MultiTaskLoss, self).__init__()
        self.task_weights = task_weights or [1.0, 1.0, 1.0]
        self.bce_loss = nn.BCELoss()
    
    def forward(self, predictions, labels):
        """
        Args:
            predictions: [task1_pred, task2_pred, ...]
            labels: [task1_label, task2_label, ...]
        """
        total_loss = 0
        
        for i, (pred, label) in enumerate(zip(predictions, labels)):
            task_loss = self.bce_loss(pred, label)
            total_loss += self.task_weights[i] * task_loss
        
        return total_loss


# 使用
criterion = MultiTaskLoss(task_weights=[1.0, 2.0, 0.5])  # CVR 权重高

# 训练
predictions = model(user_ids, item_ids)
labels = [ctr_labels, cvr_labels, duration_labels]
loss = criterion(predictions, labels)
loss.backward()
```

## 14.4 MMOE（Multi-gate Mixture-of-Experts）

### 14.4.1 核心思想

> 用多个专家网络 + 门控机制，让不同任务选择不同专家组合

### 14.4.2 架构

```
┌─────────────────────────────────────────────┐
│   任务 1 门控   │   任务 2 门控   │   任务 3 门控   │
│   (Gate 1)    │   (Gate 2)    │   (Gate 3)    │
└───────┬───────┘   └───────┬───────┘   └───────┬───────┘
        │                   │                   │
        └───────────────────┼───────────────────┘
                            │
        ┌───────────────────┼───────────────────┐
        │                   │                   │
┌───────▼───────┐   ┌───────▼───────┐   ┌───────▼───────┐
│   专家 1       │   │   专家 2       │   │   专家 3       │
│   (Expert)    │   │   (Expert)    │   │   (Expert)    │
└───────┬───────┘   └───────┬───────┘   └───────┬───────┘
        │                   │                   │
        └───────────────────┼───────────────────┘
                            │
                    ┌───────▼───────┐
                    │    嵌入层      │
                    └───────────────┘
```

### 14.4.3 代码实现

```python
class MMOE(nn.Module):
    """MMOE 多任务模型"""
    
    def __init__(
        self,
        n_users,
        n_items,
        embedding_dim=16,
        n_experts=8,
        expert_hidden_dims=[64],
        task_hidden_dims=[32],
        n_tasks=3,
    ):
        super(MMOE, self).__init__()
        
        self.n_tasks = n_tasks
        self.n_experts = n_experts
        
        # 嵌入层
        self.user_embedding = nn.Embedding(n_users, embedding_dim)
        self.item_embedding = nn.Embedding(n_items, embedding_dim)
        
        # 专家网络（多个）
        self.experts = nn.ModuleList()
        
        for _ in range(n_experts):
            expert_layers = []
            input_dim = embedding_dim * 2
            
            for hidden_dim in expert_hidden_dims:
                expert_layers.append(nn.Linear(input_dim, hidden_dim))
                expert_layers.append(nn.ReLU())
                input_dim = hidden_dim
            
            self.experts.append(nn.Sequential(*expert_layers))
        
        # 门控网络（每个任务一个）
        self.gates = nn.ModuleList()
        
        for _ in range(n_tasks):
            gate = nn.Sequential(
                nn.Linear(embedding_dim * 2, n_experts),
                nn.Softmax(dim=1)
            )
            self.gates.append(gate)
        
        # 任务塔
        self.task_towers = nn.ModuleList()
        
        for _ in range(n_tasks):
            tower_input = expert_hidden_dims[-1] if expert_hidden_dims else embedding_dim * 2
            
            tower_layers = []
            for hidden_dim in task_hidden_dims:
                tower_layers.append(nn.Linear(tower_input, hidden_dim))
                tower_layers.append(nn.ReLU())
                tower_input = hidden_dim
            
            tower_layers.append(nn.Linear(tower_input, 1))
            self.task_towers.append(nn.Sequential(*tower_layers))
    
    def forward(self, user_ids, item_ids):
        # 嵌入
        user_emb = self.user_embedding(user_ids)
        item_emb = self.item_embedding(item_ids)
        concat = torch.cat([user_emb, item_emb], dim=1)
        
        # 专家输出
        expert_outputs = []
        for expert in self.experts:
            expert_outputs.append(expert(concat))  # [batch, hidden]
        
        expert_outputs = torch.stack(expert_outputs, dim=1)  # [batch, n_experts, hidden]
        
        # 门控权重 + 专家融合
        task_outputs = []
        
        for i, gate in enumerate(self.gates):
            gate_weights = gate(concat).unsqueeze(1)  # [batch, 1, n_experts]
            
            # 加权融合专家输出
            combined = torch.matmul(gate_weights, expert_outputs).squeeze(1)  # [batch, hidden]
            
            # 任务塔
            output = self.task_towers[i](combined)
            task_outputs.append(torch.sigmoid(output).squeeze())
        
        return task_outputs


# 使用
model = MMOE(
    n_users=1000,
    n_items=500,
    embedding_dim=16,
    n_experts=8,
    expert_hidden_dims=[64],
    task_hidden_dims=[32],
    n_tasks=3
)

outputs = model(user_ids, item_ids)
```

## 14.5 PLE（Progressive Layered Extraction）

### 14.5.1 改进 MMOE

**MMOE 的问题：**
- 所有任务共享所有专家
- 任务差异大时效果下降

**PLE 的改进：**
- 分离专家（共享专家 + 任务专属专家）
- 分层提取信息

### 14.5.2 架构

```
任务 1                    任务 2                    任务 3
  │                       │                       │
  ├─ 专属专家 1           ├─ 专属专家 2           ├─ 专属专家 3
  │                       │                       │
  └───────────┬───────────┴───────────┬───────────┘
              │                       │
      ┌───────▼───────┐       ┌───────▼───────┐
      │   共享专家 1   │       │   共享专家 2   │
      └───────┬───────┘       └───────┬───────┘
              │                       │
              └───────────┬───────────┘
                          │
                  ┌───────▼───────┐
                  │    嵌入层      │
                  └───────────────┘
```

### 14.5.3 代码实现

```python
class PLE(nn.Module):
    """PLE 多任务模型"""
    
    def __init__(
        self,
        n_users,
        n_items,
        embedding_dim=16,
        n_shared_experts=4,
        n_task_experts=2,
        expert_hidden_dims=[64],
        task_hidden_dims=[32],
        n_tasks=3,
        n_layers=2,
    ):
        super(PLE, self).__init__()
        
        self.n_tasks = n_tasks
        self.n_layers = n_layers
        
        # 嵌入层
        self.user_embedding = nn.Embedding(n_users, embedding_dim)
        self.item_embedding = nn.Embedding(n_items, embedding_dim)
        
        # 多层 CGC（Customized Gate Control）
        self.cgc_layers = nn.ModuleList()
        
        input_dim = embedding_dim * 2
        
        for layer in range(n_layers):
            cgc = CGC(
                input_dim=input_dim,
                n_shared_experts=n_shared_experts,
                n_task_experts=n_task_experts,
                expert_hidden_dims=expert_hidden_dims,
                n_tasks=n_tasks,
            )
            self.cgc_layers.append(cgc)
            input_dim = expert_hidden_dims[-1] if expert_hidden_dims else embedding_dim * 2
        
        # 任务塔
        self.task_towers = nn.ModuleList()
        
        for _ in range(n_tasks):
            tower_layers = []
            tower_input = input_dim
            
            for hidden_dim in task_hidden_dims:
                tower_layers.append(nn.Linear(tower_input, hidden_dim))
                tower_layers.append(nn.ReLU())
                tower_input = hidden_dim
            
            tower_layers.append(nn.Linear(tower_input, 1))
            self.task_towers.append(nn.Sequential(*tower_layers))
    
    def forward(self, user_ids, item_ids):
        # 嵌入
        user_emb = self.user_embedding(user_ids)
        item_emb = self.item_embedding(item_ids)
        concat = torch.cat([user_emb, item_emb], dim=1)
        
        # 多层 CGC
        task_reprs = [concat] * self.n_tasks  # 每个任务的表示
        
        for cgc_layer in self.cgc_layers:
            task_reprs = cgc_layer(concat, task_reprs)
        
        # 任务塔
        task_outputs = []
        for i, tower in enumerate(self.task_towers):
            output = tower(task_reprs[i])
            task_outputs.append(torch.sigmoid(output).squeeze())
        
        return task_outputs


class CGC(nn.Module):
    """Customized Gate Control (CGC) 模块"""
    
    def __init__(
        self,
        input_dim,
        n_shared_experts,
        n_task_experts,
        expert_hidden_dims,
        n_tasks,
    ):
        super(CGC, self).__init__()
        
        self.n_tasks = n_tasks
        
        # 共享专家
        self.shared_experts = nn.ModuleList()
        for _ in range(n_shared_experts):
            expert = self._build_expert(input_dim, expert_hidden_dims)
            self.shared_experts.append(expert)
        
        # 任务专属专家
        self.task_experts = nn.ModuleList()
        for _ in range(n_tasks):
            experts = nn.ModuleList()
            for _ in range(n_task_experts):
                expert = self._build_expert(input_dim, expert_hidden_dims)
                experts.append(expert)
            self.task_experts.append(experts)
        
        # 门控
        self.task_gates = nn.ModuleList()
        for _ in range(n_tasks):
            gate_input_dim = input_dim + n_shared_experts * expert_hidden_dims[-1] + n_task_experts * expert_hidden_dims[-1]
            gate = nn.Sequential(
                nn.Linear(gate_input_dim, n_shared_experts + n_task_experts),
                nn.Softmax(dim=1)
            )
            self.task_gates.append(gate)
    
    def _build_expert(self, input_dim, hidden_dims):
        layers = []
        for hidden_dim in hidden_dims:
            layers.append(nn.Linear(input_dim, hidden_dim))
            layers.append(nn.ReLU())
            input_dim = hidden_dim
        return nn.Sequential(*layers)
    
    def forward(self, input_repr, task_reprs):
        # 共享专家输出
        shared_outputs = [expert(input_repr) for expert in self.shared_experts]
        shared_outputs = torch.cat(shared_outputs, dim=1)
        
        # 任务专属专家输出
        task_expert_outputs = []
        for i in range(self.n_tasks):
            experts_output = [expert(task_reprs[i]) for expert in self.task_experts[i]]
            task_expert_outputs.append(torch.cat(experts_output, dim=1))
        
        # 门控融合
        new_task_reprs = []
        for i in range(self.n_tasks):
            gate_input = torch.cat([input_repr, shared_outputs, task_expert_outputs[i]], dim=1)
            gate_weights = self.task_gates[i](gate_input)
            
            # 融合
            all_experts = torch.cat([shared_outputs, task_expert_outputs[i]], dim=1)
            combined = torch.sum(gate_weights.unsqueeze(1) * all_experts, dim=1)
            new_task_reprs.append(combined)
        
        return new_task_reprs
```

## 14.6 损失加权策略

### 14.6.1 等权重

```python
loss = loss_ctr + loss_cvr + loss_duration
```

### 14.6.2 手动加权

```python
loss = 1.0 * loss_ctr + 2.0 * loss_cvr + 0.5 * loss_duration
```

### 14.6.3 动态加权（Uncertainty Weighting）

```python
class DynamicWeightedLoss(nn.Module):
    """动态加权损失"""
    
    def __init__(self, n_tasks):
        super(DynamicWeightedLoss, self).__init__()
        
        # 可学习的权重参数
        self.log_vars = nn.Parameter(torch.zeros(n_tasks))
    
    def forward(self, losses):
        """
        Args:
            losses: [loss1, loss2, ...]
        """
        total_loss = 0
        
        for i, loss in enumerate(losses):
            # 精度加权
            precision = torch.exp(-self.log_vars[i])
            total_loss += precision * loss + self.log_vars[i]
        
        return total_loss


# 使用
criterion = DynamicWeightedLoss(n_tasks=3)
losses = [loss_ctr, loss_cvr, loss_duration]
total_loss = criterion(losses)
```
