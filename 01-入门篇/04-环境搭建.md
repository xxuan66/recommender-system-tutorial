```bash

# 安装时勾选 "Add Python to PATH"

# 验证安装
python --version
```

### 4.1.2 创建虚拟环境

**为什么建议使用虚拟环境？**
- 隔离项目依赖
- 避免版本冲突
- 方便部署和分享

```bash
# 创建项目目录
mkdir recommender-system-tutorial
cd recommender-system-tutorial

# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境

# macOS/Linux
source venv/bin/activate

# Windows
venv\Scripts\activate

# 验证激活成功（命令行前面应该有 (venv)）
which python  # macOS/Linux
where python  # Windows
```

## 4.3 推荐系统专用库

### 4.3.1 Surprise

**Surprise** 是一个专门面向推荐系统教学和实验的 Python 库，内置了多种经典算法。

```bash
# 安装
pip install "scikit-surprise>=1.1.0"

# 验证
python3 -c "import surprise; print(surprise.__version__)"
```

**快速示例：**
```python
from surprise import Dataset, SVD, KNNBasic
from surprise.model_selection import train_test_split
from surprise import accuracy

# 加载 MovieLens 数据集
data = Dataset.load_builtin('ml-100k')

# 划分训练集和测试集
trainset, testset = train_test_split(data, test_size=0.25)

# 使用 SVD 算法
algo = SVD()
algo.fit(trainset)

# 预测
predictions = algo.test(testset)

# 评估
accuracy.rmse(predictions)
```

### 4.3.2 Implicit

**Implicit** 更适合做隐式反馈推荐实验。

```bash
# 安装
pip install implicit>=0.5.0

# 验证
python3 -c "import implicit; print(implicit.__version__)"
```

**快速示例：**
```python
import implicit
import numpy as np

# 创建用户 - 物品交互矩阵
user_item_matrix = np.random.randint(0, 2, (100, 50))

# 使用 ALS 算法
model = implicit.als.AlternatingLeastSquares(factors=50)
model.fit(user_item_matrix)

# 为用户 0 推荐
user_id = 0
recommendations = model.recommend(user_id, user_item_matrix[user_id])
print(f"为用户 {user_id} 推荐的物品：{recommendations[:5]}")
```

### 4.3.3 LightFM

**LightFM** 适合做混合推荐（协同过滤 + 内容特征）实验。

```bash
# 安装
pip install lightfm>=1.16

# 验证
python3 -c "from lightfm import LightFM; print('LightFM 安装成功')"
```

**快速示例：**
```python
from lightfm import LightFM
from lightfm.datasets import fetch_movielens

# 加载数据
data = fetch_movielens()

# 创建模型
model = LightFM(loss='warp')

# 训练
model.fit(data['train'], epochs=30)

# 评估
from lightfm.evaluation import precision_at_k
precision = precision_at_k(model, data['test'], k=5)
print(f"Precision@5: {precision.mean():.4f}")
```

**下载方式：**
```python
# 方式 1：使用 Surprise 自动下载
from surprise import Dataset

import urllib.request

url = "https://files.grouplens.org/datasets/movielens/ml-latest-small.zip"
urllib.request.urlretrieve(url, "ml-latest-small.zip")

# 解压
import zipfile
with zipfile.ZipFile("ml-latest-small.zip", "r") as zip_ref:
    zip_ref.extractall(".")
```

## 4.9 第一个推荐系统示例

### 4.9.1 使用 Surprise 实现协同过滤

创建 `first_recommender.py`：

```python
from surprise import Dataset, KNNBasic
from surprise.model_selection import train_test_split
from surprise import accuracy

# 1. 加载数据
print("📊 加载 MovieLens 数据集...")
data = Dataset.load_builtin('ml-100k')

# 2. 划分训练集和测试集
print("📝 划分数据集...")
trainset, testset = train_test_split(data, test_size=0.25, random_state=42)

# 3. 训练模型（ItemCF 算法）
print("🤖 训练 ItemCF 模型...")
algo = KNNBasic(sim_options={'name': 'cosine', 'user_based': False})
algo.fit(trainset)

# 4. 预测
print("🔮 进行预测...")
predictions = algo.test(testset)

# 5. 评估
print("📈 评估结果...")
accuracy.rmse(predictions)
accuracy.mae(predictions)

# 6. 为用户推荐
print("\n🎬 为用户 1 推荐电影...")
user_id = 1
# 获取用户未评分的电影
all_items = algo.trainset.known_items
user_rated = [iid for (uid, iid, _) in algo.trainset.ur[user_id]]
unrated = [iid for iid in all_items if iid not in user_rated]

# 预测评分
recommendations = []
for iid in unrated[:10]:
    est = algo.predict(user_id, iid).est
    recommendations.append((iid, est))

# 排序并输出
recommendations.sort(key=lambda x: x[1], reverse=True)
print(f"为用户 1 推荐的 Top 5 电影：")
for iid, score in recommendations[:5]:
    print(f"  电影 {iid}: 预测评分 {score:.2f}")

print("\n✅ 第一个推荐系统完成！")
```

**运行：**
```bash
python3 first_recommender.py
```

## 4.10 常见问题


### Q3: 依赖冲突

**解决方案：**
```bash
# 升级 pip
pip install --upgrade pip

# 清除缓存
pip cache purge

# 重新安装
pip install -r requirements.txt --force-reinstall
```

### Q4: CUDA 版本不匹配

**解决方案：**
```bash
# 查看当前 CUDA 版本
nvcc --version

# 访问 https://pytorch.org 获取对应版本的安装命令
# 例如 CUDA 11.1:

```
