# 第七章：矩阵分解

> 通过降维技术发现用户和物品的隐含特征

---

## 7.1 什么是矩阵分解

### 7.1.1 核心思想

**矩阵分解（Matrix Factorization）** 的基本思想是：

> 将用户 - 物品评分矩阵分解为两个低维矩阵的乘积，发现用户和物品的隐含特征。

### 7.1.2 数学表示

```
评分矩阵 R ≈ P × Q^T

其中：
- R: m×n 评分矩阵（m 个用户，n 个物品）
- P: m×k 用户隐含特征矩阵
- Q: n×k 物品隐含特征矩阵
- k: 隐含特征维度（k << m, k << n）
```

**图示：**
```
R (m×n)     ≈     P (m×k)    ×    Q^T (k×n)

[用户 1]  [5  3  0  4]           [0.5  0.3]    [0.8  0.2  0.1  0.9]
[用户 2]  [4  0  3  5]     ≈     [0.3  0.7]  × [0.6  0.8  0.4  0.5]
[用户 3]  [0  4  5  0]           [0.9  0.1]    [0.2  0.9  0.7  0.3]
                                ↑             ↑
                            用户特征       物品特征
                            (动作/爱情)   (动作/爱情)
```

### 7.1.3 为什么有效

**隐含特征的例子（电影推荐）：**

| 隐含特征 1 | 隐含特征 2 | 隐含特征 3 |
|-----------|-----------|-----------|
| 动作程度 | 爱情程度 | 科幻程度 |
| 喜剧程度 | 剧情程度 | 惊悚程度 |
| 商业程度 | 艺术程度 | 经典程度 |

**用户和物品都可以用这些特征表示：**
- 用户 A：[0.8, 0.2, 0.6] — 喜欢动作、科幻
- 电影 X：[0.9, 0.1, 0.8] — 动作科幻片
- 预测评分：0.8×0.9 + 0.2×0.1 + 0.6×0.8 = 1.22

## 7.2 SVD（奇异值分解）

### 7.2.1 原理

**SVD 分解公式：**
```
R = U × Σ × V^T

其中：
- U: m×m 正交矩阵（左奇异向量）
- Σ: m×n 对角矩阵（奇异值）
- V: n×n 正交矩阵（右奇异向量）
```

**降维近似：**
```
R ≈ U_k × Σ_k × V_k^T

其中 k 是前 k 个最大的奇异值
```

### 7.2.2 代码实现

```python
import numpy as np
from scipy.sparse.linalg import svds

class SVDRecommender:
    """基于 SVD 的推荐系统"""
    
    def __init__(self, k=10):
        """
        Args:
            k: 隐含特征维度
        """
        self.k = k
        
    def fit(self, ratings):
        """
        训练模型
        
        Args:
            ratings: 用户 - 物品评分矩阵
        """
        self.ratings = ratings
        
        # 填充缺失值（用全局平均分）
        self.global_mean = np.mean(ratings[ratings > 0])
        ratings_filled = ratings.copy()
        ratings_filled[ratings == 0] = self.global_mean
        
        # SVD 分解
        U, sigma, Vt = svds(ratings_filled, k=self.k)
        
        # 保存分解结果
        self.U = U
        self.sigma = np.diag(sigma)
        self.Vt = Vt
        
        return self
    
    def predict(self, u, i):
        """预测评分"""
        pred = np.dot(np.dot(self.U[u], self.sigma), self.Vt[:, i])
        return pred
    
    def recommend(self, u, n=10):
        """推荐 Top-N 物品"""
        n_items = self.ratings.shape[1]
        
        # 预测所有物品评分
        predictions = []
        for i in range(n_items):
            if self.ratings[u, i] == 0:  # 未评分
                score = self.predict(u, i)
                predictions.append((i, score))
        
        # 排序
        predictions.sort(key=lambda x: x[1], reverse=True)
        return predictions[:n]


# 使用示例
if __name__ == "__main__":
    ratings = np.array([
        [5, 3, 0, 4, 0],
        [4, 0, 3, 5, 2],
        [0, 4, 5, 0, 3],
        [3, 0, 4, 0, 5],
        [0, 5, 0, 3, 4],
    ])
    
    model = SVDRecommender(k=3)
    model.fit(ratings)
    
    user_id = 0
    recommendations = model.recommend(user_id, n=3)
    
    print(f"为用户 {user_id} 推荐的物品：")
    for item_id, score in recommendations:
        print(f"  物品 {item_id}: 预测评分 {score:.2f}")
```

### 7.2.3 SVD 的局限性

| 问题 | 说明 |
|------|------|
| **需要稠密矩阵** | SVD 要求矩阵是稠密的，需要填充缺失值 |
| **无法处理稀疏数据** | 填充会引入噪声 |
| **计算复杂度高** | O(m×n×k)，大数据集慢 |
| **无法增量更新** | 新数据需要重新分解 |

## 7.3  FunkSVD

### 7.3.1 改进思路

**FunkSVD（Simon Funk 提出）** 的核心改进：

> 不填充缺失值，只优化已知的评分

### 7.3.2 优化目标

```
最小化：Σ(u,i)∈R (ru,i - pu^T × qi)^2 + λ(||pu||^2 + ||qi||^2)

其中：
- ru,i: 用户 u 对物品 i 的真实评分
- pu: 用户 u 的隐含特征向量
- qi: 物品 i 的隐含特征向量
- λ: 正则化参数（防止过拟合）
```

### 7.3.3 梯度下降优化

```python
class FunkSVD:
    """FunkSVD 实现"""
    
    def __init__(self, k=10, lr=0.01, reg=0.02, epochs=100):
        """
        Args:
            k: 隐含特征维度
            lr: 学习率
            reg: 正则化参数
            epochs: 训练轮数
        """
        self.k = k
        self.lr = lr
        self.reg = reg
        self.epochs = epochs
        
    def fit(self, ratings):
        """训练模型"""
        self.ratings = ratings
        n_users, n_items = ratings.shape
        
        # 随机初始化 P 和 Q
        np.random.seed(42)
        self.P = np.random.normal(0, 0.1, (n_users, self.k))
        self.Q = np.random.normal(0, 0.1, (n_items, self.k))
        
        # 全局偏置和用户/物品偏置
        self.global_mean = np.mean(ratings[ratings > 0])
        self.bu = np.zeros(n_users)  # 用户偏置
        self.bi = np.zeros(n_items)  # 物品偏置
        
        # 找到所有已知评分
        known_ratings = []
        for u in range(n_users):
            for i in range(n_items):
                if ratings[u, i] > 0:
                    known_ratings.append((u, i, ratings[u, i]))
        
        # 梯度下降
        for epoch in range(self.epochs):
            np.random.shuffle(known_ratings)
            total_error = 0
            
            for u, i, r in known_ratings:
                # 预测
                pred = self.global_mean + self.bu[u] + self.bi[i] + np.dot(self.P[u], self.Q[i])
                
                # 误差
                error = r - pred
                total_error += error ** 2
                
                # 更新偏置
                self.bu[u] += self.lr * (error - self.reg * self.bu[u])
                self.bi[i] += self.lr * (error - self.reg * self.bi[i])
                
                # 更新隐含特征
                pu_old = self.P[u].copy()
                self.P[u] += self.lr * (error * self.Q[i] - self.reg * self.P[u])
                self.Q[i] += self.lr * (error * pu_old - self.reg * self.Q[i])
            
            # 打印训练进度
            if (epoch + 1) % 20 == 0:
                rmse = np.sqrt(total_error / len(known_ratings))
                print(f"Epoch {epoch+1}/{self.epochs}, RMSE: {rmse:.4f}")
        
        return self
    
    def predict(self, u, i):
        """预测评分"""
        pred = self.global_mean + self.bu[u] + self.bi[i] + np.dot(self.P[u], self.Q[i])
        return np.clip(pred, 0.5, 5.0)  # 限制在评分范围内
    
    def recommend(self, u, n=10):
        """推荐 Top-N 物品"""
        n_items = self.ratings.shape[1]
        
        predictions = []
        for i in range(n_items):
            if self.ratings[u, i] == 0:
                score = self.predict(u, i)
                predictions.append((i, score))
        
        predictions.sort(key=lambda x: x[1], reverse=True)
        return predictions[:n]


# 使用示例
if __name__ == "__main__":
    ratings = np.array([
        [5, 3, 0, 4, 0],
        [4, 0, 3, 5, 2],
        [0, 4, 5, 0, 3],
        [3, 0, 4, 0, 5],
        [0, 5, 0, 3, 4],
    ])
    
    model = FunkSVD(k=5, lr=0.01, reg=0.02, epochs=100)
    model.fit(ratings)
    
    user_id = 0
    recommendations = model.recommend(user_id, n=3)
    
    print(f"\n为用户 {user_id} 推荐的物品：")
    for item_id, score in recommendations:
        print(f"  物品 {item_id}: 预测评分 {score:.2f}")
```

## 7.4 SVD++

### 7.4.1 改进思路

**SVD++** 在 FunkSVD 基础上增加了：

> 考虑用户的隐式反馈（浏览、点击等行为）

### 7.4.2 预测公式

```
P(u, i) = μ + bu + bi + qi^T × (pu + |Iu|^(-0.5) × Σ(j∈Iu) yj)

其中：
- Iu: 用户 u 有过隐式反馈的物品集合
- yj: 物品 j 的隐式反馈向量
- |Iu|^(-0.5): 归一化因子
```

### 7.4.3 代码实现

```python
class SVDPlusPlus:
    """SVD++ 实现"""
    
    def __init__(self, k=10, lr=0.01, reg=0.02, epochs=100):
        self.k = k
        self.lr = lr
        self.reg = reg
        self.epochs = epochs
        
    def fit(self, ratings, implicit_feedback=None):
        """
        训练模型
        
        Args:
            ratings: 显式评分矩阵
            implicit_feedback: 隐式反馈矩阵（0/1）
        """
        self.ratings = ratings
        n_users, n_items = ratings.shape
        
        # 隐式反馈（如果没有，用评分矩阵代替）
        if implicit_feedback is None:
            self.implicit = (ratings > 0).astype(float)
        else:
            self.implicit = implicit_feedback
        
        # 初始化参数
        np.random.seed(42)
        self.P = np.random.normal(0, 0.1, (n_users, self.k))
        self.Q = np.random.normal(0, 0.1, (n_items, self.k))
        self.Y = np.random.normal(0, 0.1, (n_items, self.k))  # 隐式反馈向量
        
        self.global_mean = np.mean(ratings[ratings > 0])
        self.bu = np.zeros(n_users)
        self.bi = np.zeros(n_items)
        
        # 找到已知评分
        known_ratings = []
        for u in range(n_users):
            for i in range(n_items):
                if ratings[u, i] > 0:
                    known_ratings.append((u, i, ratings[u, i]))
        
        # 梯度下降
        for epoch in range(self.epochs):
            np.random.shuffle(known_ratings)
            total_error = 0
            
            for u, i, r in known_ratings:
                # 用户隐式反馈集合
                Iu = np.where(self.implicit[u] > 0)[0]
                
                # 隐式反馈的平均向量
                if len(Iu) > 0:
                    y_u = np.mean(self.Y[Iu], axis=0)
                else:
                    y_u = np.zeros(self.k)
                
                # 预测
                pred = self.global_mean + self.bu[u] + self.bi[i] + \
                       np.dot(self.Q[i], self.P[u] + y_u)
                
                error = r - pred
                total_error += error ** 2
                
                # 更新参数
                self.bu[u] += self.lr * (error - self.reg * self.bu[u])
                self.bi[i] += self.lr * (error - self.reg * self.bi[i])
                
                pu_old = self.P[u].copy()
                self.P[u] += self.lr * (error * self.Q[i] - self.reg * self.P[u])
                self.Q[i] += self.lr * (error * (pu_old + y_u) - self.reg * self.Q[i])
                
                # 更新隐式反馈向量
                if len(Iu) > 0:
                    for j in Iu:
                        self.Y[j] += self.lr * (error * self.Q[i] / len(Iu) - self.reg * self.Y[j])
            
            if (epoch + 1) % 20 == 0:
                rmse = np.sqrt(total_error / len(known_ratings))
                print(f"Epoch {epoch+1}/{self.epochs}, RMSE: {rmse:.4f}")
        
        return self
    
    def predict(self, u, i):
        """预测评分"""
        Iu = np.where(self.implicit[u] > 0)[0]
        
        if len(Iu) > 0:
            y_u = np.mean(self.Y[Iu], axis=0)
        else:
            y_u = np.zeros(self.k)
        
        pred = self.global_mean + self.bu[u] + self.bi[i] + \
               np.dot(self.Q[i], self.P[u] + y_u)
        
        return np.clip(pred, 0.5, 5.0)
    
    def recommend(self, u, n=10):
        """推荐 Top-N 物品"""
        n_items = self.ratings.shape[1]
        
        predictions = []
        for i in range(n_items):
            if self.ratings[u, i] == 0:
                score = self.predict(u, i)
                predictions.append((i, score))
        
        predictions.sort(key=lambda x: x[1], reverse=True)
        return predictions[:n]
```

## 7.5 使用 Surprise 库

### 7.5.1 SVD

```python
from surprise import SVD, Dataset, train_test_split, accuracy

# 加载数据
data = Dataset.load_builtin('ml-100k')
trainset, testset = train_test_split(data, test_size=0.25)

# 创建 SVD 模型
model = SVD(
    n_factors=100,      # 隐含特征维度
    n_epochs=50,        # 训练轮数
    lr_all=0.005,       # 学习率
    reg_all=0.02,       # 正则化参数
)

# 训练
model.fit(trainset)

# 预测
predictions = model.test(testset)

# 评估
accuracy.rmse(predictions)
```

## 7.6 矩阵分解对比

| 算法 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| **SVD** | 简单、理论基础强 | 需要稠密矩阵 | 小规模数据 |
| **FunkSVD** | 处理稀疏数据 | 需要调参 | 一般推荐 |
| **SVD++** | 利用隐式反馈 | 计算复杂 | 有隐式反馈数据 |
