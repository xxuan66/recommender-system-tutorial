## 5.1 什么是协同过滤

### 5.1.1 核心思想

**协同过滤（Collaborative Filtering, CF）** 的基本思想是：

> 利用群体的智慧，找到相似的用户或物品，基于相似性进行推荐。

### 5.1.2 两种主要类型

```
协同过滤
├── 基于用户的协同过滤（UserCF）
│   └── "和你相似的人也喜欢 X"
└── 基于物品的协同过滤（ItemCF）
    └── "喜欢 A 的人也喜欢 B"
```

### 5.1.3 直观例子

#### UserCF 示例

```
用户 A 和用户 B 的观影记录很相似：
- 用户 A：喜欢《流浪地球》《战狼 2》《红海行动》
- 用户 B：喜欢《流浪地球》《战狼 2》《红海行动》《长津湖》

→ 推荐《长津湖》给用户 A
```

#### ItemCF 示例

```
物品相似度：
- 喜欢《流浪地球》的人也喜欢《上海堡垒》
- 用户 A 喜欢《流浪地球》

→ 推荐《上海堡垒》给用户 A
```

## 5.2 相似度计算

协同过滤的核心是计算相似度，常用的相似度计算方法有：

### 5.2.1 余弦相似度（Cosine Similarity）

**定义：** 两个向量夹角的余弦值

**公式：**
```
similarity = cos(θ) = (A·B) / (||A|| × ||B||)
```

**代码实现：**
```python
import numpy as np

def cosine_similarity(a, b):
    """
    计算余弦相似度
    
    Args:
        a, b: 两个向量
    
    Returns:
        余弦相似度值 [-1, 1]
    """
    a = np.array(a)
    b = np.array(b)
    
    dot_product = np.dot(a, b)
    norm_a = np.linalg.norm(a)
    norm_b = np.linalg.norm(b)
    
    if norm_a == 0 or norm_b == 0:
        return 0
    
    return dot_product / (norm_a * norm_b)

# 示例
user_a = [5, 3, 0, 4, 0]  # 用户对 5 个物品的评分
user_b = [4, 0, 3, 5, 2]

sim = cosine_similarity(user_a, user_b)
print(f"余弦相似度：{sim:.4f}")  # 0.7559
```

**特点：**
- 取值范围：[-1, 1]
- 值越接近 1，相似度越高
- 忽略向量长度，只关注方向

### 5.2.2 皮尔逊相关系数（Pearson Correlation）

**定义：** 两个变量的线性相关程度

**公式：**
```
similarity = Σ((ra - r̄a)(rb - r̄b)) / (√Σ(ra - r̄a)² × √Σ(rb - r̄b)²)
```

**代码实现：**
```python
def pearson_similarity(a, b):
    """
    计算皮尔逊相关系数
    """
    a = np.array(a)
    b = np.array(b)
    
    # 只计算共同评分的物品
    mask = (a > 0) & (b > 0)
    a = a[mask]
    b = b[mask]
    
    if len(a) == 0:
        return 0
    
    # 计算均值
    mean_a = np.mean(a)
    mean_b = np.mean(b)
    
    # 计算分子和分母
    numerator = np.sum((a - mean_a) * (b - mean_b))
    denominator = np.sqrt(np.sum((a - mean_a) ** 2)) * np.sqrt(np.sum((b - mean_b) ** 2))
    
    if denominator == 0:
        return 0
    
    return numerator / denominator

# 示例
user_a = [5, 3, 0, 4, 0]
user_b = [4, 0, 3, 5, 2]

sim = pearson_similarity(user_a, user_b)
print(f"皮尔逊相关系数：{sim:.4f}")  # 0.5571
```

**特点：**
- 取值范围：[-1, 1]
- 考虑了用户评分偏置（有的用户习惯打高分）
- 更适合评分数据

### 5.2.3 杰卡德相似度（Jaccard Similarity）

**定义：** 两个集合的交集与并集的比值

**公式：**
```
similarity = |A ∩ B| / |A ∪ B|
```

**代码实现：**
```python
def jaccard_similarity(a, b):
    """
    计算杰卡德相似度（适用于二元数据）
    """
    set_a = set(i for i, x in enumerate(a) if x > 0)
    set_b = set(i for i, x in enumerate(b) if x > 0)
    
    intersection = len(set_a & set_b)
    union = len(set_a | set_b)
    
    if union == 0:
        return 0
    
    return intersection / union

# 示例
user_a = [1, 1, 0, 1, 0]  # 1 表示喜欢/购买，0 表示无行为
user_b = [1, 0, 1, 1, 1]

sim = jaccard_similarity(user_a, user_b)
print(f"杰卡德相似度：{sim:.4f}")  # 0.3333
```

**特点：**
- 适用于二元数据（点击/未点击、购买/未购买）
- 忽略负反馈

### 5.2.4 相似度对比

| 相似度 | 适用场景 | 优点 | 缺点 |
|--------|---------|------|------|
| **余弦** | 一般场景 | 计算简单 | 忽略评分偏置 |
| **皮尔逊** | 评分数据 | 考虑评分偏置 | 计算复杂 |
| **杰卡德** | 隐式反馈 | 适合二元数据 | 忽略评分强度 |

## 5.3 UserCF 算法

### 5.3.1 算法原理

**UserCF 的两步：**

1. **找相似用户** — 找到和目标用户兴趣相似的其他用户
2. **推荐物品** — 将相似用户喜欢的物品推荐给目标用户

**图示：**
```
用户 A（目标用户）
  ↓ 找到相似用户
用户 B、用户 C
  ↓ 推荐他们喜欢但 A 未接触的物品
物品 X、物品 Y → 推荐给用户 A
```

### 5.3.2 算法步骤

```
输入：用户 - 物品评分矩阵 R，目标用户 u，邻居数 K
输出：推荐列表

步骤：
1. 计算用户 u 与其他所有用户的相似度
2. 选择相似度最高的 K 个用户（K 近邻）
3. 预测 u 对未评分物品的评分
4. 按预测评分排序，输出 Top-N 推荐
```

### 5.3.3 评分预测公式

```
P(u, i) = r̄u + Σ(v∈N(u)) sim(u, v) × (rv,i - r̄v) / Σ|sim(u, v)|
```

**说明：**
- `P(u, i)`：用户 u 对物品 i 的预测评分
- `r̄u`：用户 u 的平均评分
- `N(u)`：用户 u 的 K 近邻
- `sim(u, v)`：用户 u 和 v 的相似度
- `rv,i`：用户 v 对物品 i 的评分

### 5.3.4 完整代码实现

```python
import numpy as np
from collections import defaultdict

class UserCF:
    """基于用户的协同过滤"""
    
    def __init__(self, k=10):
        """
        Args:
            k: 近邻数量
        """
        self.k = k
        self.user_sim = {}  # 用户相似度矩阵
        self.user_means = {}  # 用户平均评分
        
    def fit(self, ratings):
        """
        训练模型
        
        Args:
            ratings: 用户 - 物品评分矩阵 (numpy array)
        """
        self.ratings = ratings
        n_users, n_items = ratings.shape
        
        # 计算用户平均评分
        for u in range(n_users):
            user_ratings = ratings[u][ratings[u] > 0]
            self.user_means[u] = np.mean(user_ratings) if len(user_ratings) > 0 else 0
        
        # 计算用户相似度矩阵
        self.user_sim = np.zeros((n_users, n_users))
        for i in range(n_users):
            for j in range(i, n_users):
                sim = self._pearson(i, j)
                self.user_sim[i, j] = sim
                self.user_sim[j, i] = sim
        
        return self
    
    def _pearson(self, u, v):
        """计算用户 u 和 v 的皮尔逊相似度"""
        # 找到共同评分的物品
        mask = (self.ratings[u] > 0) & (self.ratings[v] > 0)
        if np.sum(mask) == 0:
            return 0
        
        ru = self.ratings[u][mask]
        rv = self.ratings[v][mask]
        
        # 皮尔逊相关系数
        mean_u = np.mean(ru)
        mean_v = np.mean(rv)
        
        numerator = np.sum((ru - mean_u) * (rv - mean_v))
        denominator = np.sqrt(np.sum((ru - mean_u) ** 2)) * np.sqrt(np.sum((rv - mean_v) ** 2))
        
        if denominator == 0:
            return 0
        
        return numerator / denominator
    
    def predict(self, u, i):
        """
        预测用户 u 对物品 i 的评分
        
        Args:
            u: 用户 ID
            i: 物品 ID
        
        Returns:
            预测评分
        """
        # 如果用户已经评分，直接返回
        if self.ratings[u, i] > 0:
            return self.ratings[u, i]
        
        # 找到对物品 i 评分过的用户
        rated_users = np.where(self.ratings[:, i] > 0)[0]
        if len(rated_users) == 0:
            return self.user_means[u]
        
        # 获取相似度并排序
        similarities = [(v, self.user_sim[u, v]) for v in rated_users]
        similarities.sort(key=lambda x: x[1], reverse=True)
        
        # 取 K 个最近邻
        neighbors = similarities[:self.k]
        
        # 加权预测
        numerator = 0
        denominator = 0
        for v, sim in neighbors:
            if sim > 0:
                numerator += sim * (self.ratings[v, i] - self.user_means[v])
                denominator += abs(sim)
        
        if denominator == 0:
            return self.user_means[u]
        
        prediction = self.user_means[u] + numerator / denominator
        return prediction
    
    def recommend(self, u, n=10):
        """
        为用户 u 推荐 n 个物品
        
        Args:
            u: 用户 ID
            n: 推荐数量
        
        Returns:
            推荐列表 [(item_id, predicted_score), ...]
        """
        n_items = self.ratings.shape[1]
        
        # 预测所有未评分物品
        predictions = []
        for i in range(n_items):
            if self.ratings[u, i] == 0:  # 未评分
                score = self.predict(u, i)
                predictions.append((i, score))
        
        # 排序并返回 Top-N
        predictions.sort(key=lambda x: x[1], reverse=True)
        return predictions[:n]


# 使用示例
if __name__ == "__main__":
    # 创建示例数据（5 个用户对 6 个物品的评分）
    ratings = np.array([
        [5, 3, 0, 4, 0, 2],
        [4, 0, 3, 5, 2, 0],
        [0, 4, 5, 0, 3, 4],
        [3, 0, 4, 0, 5, 0],
        [0, 5, 0, 3, 4, 5],
    ])
    
    # 训练模型
    model = UserCF(k=3)
    model.fit(ratings)
    
    # 为用户 0 推荐
    user_id = 0
    recommendations = model.recommend(user_id, n=3)
    
    print(f"为用户 {user_id} 推荐的物品：")
    for item_id, score in recommendations:
        print(f"  物品 {item_id}: 预测评分 {score:.2f}")
```

```
  物品 4: 预测评分 4.52
  物品 2: 预测评分 3.85
  物品 1: 预测评分 3.21
```

## 5.4 ItemCF 算法

### 5.4.1 算法原理

**ItemCF 的两步：**

1. **找相似物品** — 找到和目标物品相似的其他物品
2. **推荐物品** — 基于用户历史喜欢的物品，推荐相似物品

**图示：**
```
用户 A 喜欢物品 X
  ↓ 找到与 X 相似的物品
物品 Y、物品 Z
  ↓ 推荐给用户 A
```

### 5.4.2 与 UserCF 的对比

| 维度 | UserCF | ItemCF |
|------|--------|--------|
| **相似度计算** | 用户之间 | 物品之间 |
| **推荐逻辑** | 相似用户喜欢的 | 相似物品的 |
| **适用场景** | 用户少、物品多 | 用户多、物品少 |
| **实时性** | 用户兴趣变化快 | 物品关系相对稳定 |
| **可解释性** | "和你相似的人也喜欢" | "喜欢这个的人也喜欢" |

### 5.4.3 评分预测公式

```
P(u, i) = Σ(j∈N(i)) sim(i, j) × ru,j / Σ|sim(i, j)|
```

**说明：**
- `P(u, i)`：用户 u 对物品 i 的预测评分
- `N(i)`：与物品 i 最相似的 K 个物品
- `sim(i, j)`：物品 i 和 j 的相似度
- `ru,j`：用户 u 对物品 j 的评分

### 5.4.4 完整代码实现

```python
import numpy as np

class ItemCF:
    """基于物品的协同过滤"""
    
    def __init__(self, k=10):
        """
        Args:
            k: 近邻数量
        """
        self.k = k
        self.item_sim = {}  # 物品相似度矩阵
        
    def fit(self, ratings):
        """
        训练模型
        
        Args:
            ratings: 用户 - 物品评分矩阵 (numpy array)
        """
        self.ratings = ratings
        n_users, n_items = ratings.shape
        
        # 计算物品相似度矩阵
        self.item_sim = np.zeros((n_items, n_items))
        for i in range(n_items):
            for j in range(i, n_items):
                sim = self._cosine(i, j)
                self.item_sim[i, j] = sim
                self.item_sim[j, i] = sim
        
        return self
    
    def _cosine(self, i, j):
        """计算物品 i 和 j 的余弦相似度"""
        # 找到对两个物品都评分的用户
        mask = (self.ratings[:, i] > 0) & (self.ratings[:, j] > 0)
        if np.sum(mask) == 0:
            return 0
        
        ri = self.ratings[mask, i]
        rj = self.ratings[mask, j]
        
        # 余弦相似度
        dot_product = np.dot(ri, rj)
        norm_i = np.linalg.norm(ri)
        norm_j = np.linalg.norm(rj)
        
        if norm_i == 0 or norm_j == 0:
            return 0
        
        return dot_product / (norm_i * norm_j)
    
    def predict(self, u, i):
        """
        预测用户 u 对物品 i 的评分
        """
        # 如果用户已经评分，直接返回
        if self.ratings[u, i] > 0:
            return self.ratings[u, i]
        
        # 找到用户 u 评分过的物品
        rated_items = np.where(self.ratings[u] > 0)[0]
        if len(rated_items) == 0:
            return 0
        
        # 获取与物品 i 的相似度并排序
        similarities = [(j, self.item_sim[i, j]) for j in rated_items]
        similarities.sort(key=lambda x: x[1], reverse=True)
        
        # 取 K 个最相似物品
        neighbors = similarities[:self.k]
        
        # 加权预测
        numerator = 0
        denominator = 0
        for j, sim in neighbors:
            if sim > 0:
                numerator += sim * self.ratings[u, j]
                denominator += abs(sim)
        
        if denominator == 0:
            return 0
        
        return numerator / denominator
    
    def recommend(self, u, n=10):
        """
        为用户 u 推荐 n 个物品
        """
        n_items = self.ratings.shape[1]
        
        # 预测所有未评分物品
        predictions = []
        for i in range(n_items):
            if self.ratings[u, i] == 0:
                score = self.predict(u, i)
                predictions.append((i, score))
        
        # 排序并返回 Top-N
        predictions.sort(key=lambda x: x[1], reverse=True)
        return predictions[:n]


# 使用示例
if __name__ == "__main__":
    # 创建示例数据
    ratings = np.array([
        [5, 3, 0, 4, 0, 2],
        [4, 0, 3, 5, 2, 0],
        [0, 4, 5, 0, 3, 4],
        [3, 0, 4, 0, 5, 0],
        [0, 5, 0, 3, 4, 5],
    ])
    
    # 训练模型
    model = ItemCF(k=3)
    model.fit(ratings)
    
    # 为用户 0 推荐
    user_id = 0
    recommendations = model.recommend(user_id, n=3)
    
    print(f"为用户 {user_id} 推荐的物品：")
    for item_id, score in recommendations:
        print(f"  物品 {item_id}: 预测评分 {score:.2f}")
```

## 5.6 优化技巧

### 5.6.1 数据归一化

**问题：** 不同用户评分标准不同（有人习惯打高分，有人习惯打低分）

**解决方案：** 均值中心化

```python
# 用户均值中心化
def normalize_user_ratings(ratings):
    normalized = ratings.copy()
    for u in range(ratings.shape[0]):
        user_ratings = ratings[u][ratings[u] > 0]
        if len(user_ratings) > 0:
            mean = np.mean(user_ratings)
            normalized[u][ratings[u] > 0] = ratings[u][ratings[u] > 0] - mean
    return normalized
```

### 5.6.2 惩罚热门物品

**问题：** 热门物品与很多物品都有相似度，导致推荐缺乏针对性

**解决方案：** 惩罚热门物品

```python
def adjusted_cosine(ratings, i, j):
    """
    调整的余弦相似度（惩罚热门物品）
    """
    # 找到对两个物品都评分的用户
    mask = (ratings[:, i] > 0) & (ratings[:, j] > 0)
    if np.sum(mask) == 0:
        return 0
    
    ri = ratings[mask, i]
    rj = ratings[mask, j]
    
    # 余弦相似度
    dot_product = np.dot(ri, rj)
    norm_i = np.linalg.norm(ri)
    norm_j = np.linalg.norm(rj)
    
    if norm_i == 0 or norm_j == 0:
        return 0
    
    # 惩罚因子（共同评分用户数越少，惩罚越大）
    penalty = np.sum(mask) / ratings.shape[0]
    
    return (dot_product / (norm_i * norm_j)) * penalty
```

### 5.6.3 相似度阈值

**问题：** 有些用户/物品相似度很低，但也被纳入计算

**解决方案：** 设置相似度阈值

```python
# 只使用相似度 > 0.5 的近邻
neighbors = [(v, sim) for v, sim in similarities if sim > 0.5]
```
